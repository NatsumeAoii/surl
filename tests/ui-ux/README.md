# UI/UX Tests

Use this folder for UI and UX verification assets that are not part of the unit test suite.

Recommended contents:

- Accessibility check notes.
- Keyboard navigation test cases.
- Responsive viewport screenshots or reports.
- Visual regression baselines and review notes.
- Manual QA matrices for loading, empty, error, success, and disabled states.

Do not commit generated reports, screenshots with private data, local browser profiles, or temporary test output.
